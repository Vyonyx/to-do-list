# to-do-list
Creating a to do list app that is able to create to do items. These items should be able to store a title, description, due date and priority (3x options).
A to do item should be stored inside a project. By default, it can go into a default project, however, a user should be able to choose if they want to create an new project folder for a new to-do item.
The interface should be simple at a glance while being able to unveil a more detailed window.
